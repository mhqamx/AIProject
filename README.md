# AIProject
AI测试工程 - Vue 3 实践项目

## 项目介绍

这是一个用于AI探索和学习的代码仓库,使用Vue 3框架进行前端开发。

## 当前项目

### 📝 TodoList应用 (`/todo-app`)

功能完整的待办事项管理应用,使用Vue 3 Composition API开发。

**主要功能:**
- ✅ 添加新任务
- ✅ 标记任务完成/取消完成
- ✅ 删除任务
- ✅ 实时统计(总计、已完成、未完成)
- ✅ 响应式设计
- ✅ 流畅的动画效果

**技术栈:**
- Vue 3 (Composition API)
- Vite
- JavaScript

## 快速开始

```bash
# 进入项目目录
cd todo-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
